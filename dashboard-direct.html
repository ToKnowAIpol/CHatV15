Debugging Instructions

If you're still having issues accessing the dashboard, try the following:

  1. Open your browser's developer console (F12 or Ctrl+Shift+I)
  2. Copy and paste the following code into the console:
// Check if you have an active session
const checkSession = async () => {
  try {
    const { data, error } = await supabase.auth.getSession();
    console.log('Session check result:', data, error);
    return data.session;
  } catch (e) {
    console.error('Error checking session:', e);
    return null;
  }
};

// Try to navigate to dashboard
const goToDashboard = () => {
  window.location.href = '/dashboard';
};

// Run the check
(async () => {
  const session = await checkSession();
  if (session) {
    console.log('Active session found, redirecting to dashboard...');
    goToDashboard();
  } else {
    console.log('No active session found. Try signing in first.');
  }
})();

This will check if you have an active session and attempt to navigate to the dashboard.
